# Strona Hoff-Med — instrukcja

Strona internetowa szkoleń z pierwszej pomocy (Jakub Hoffmann, Hoff-Med).

## Pliki w projekcie

- `index.html` — treść strony (teksty, sekcje)
- `styles.css` — wygląd (kolory, układ)
- `script.js` — drobne funkcje (menu na telefonie)
- `images/` — folder na logo i zdjęcia

## Jak podejrzeć stronę na swoim komputerze

Kliknij dwukrotnie plik `index.html` — otworzy się w przeglądarce.
Po każdej zmianie w plikach wystarczy odświeżyć stronę (klawisz F5).

## Jak zmienić teksty

Otwórz `index.html` w dowolnym edytorze (np. Notatnik, albo lepiej VS Code)
i zmień tekst między znacznikami. Reszta zadziała sama.

## Jak dodać zdjęcia

Wrzuć pliki do folderu `images/` zgodnie z instrukcją w pliku
`images/CZYTAJ-MNIE.txt` (logo.png, jakub.jpg, hero.jpg).

## Jak zmienić kolory

W pliku `styles.css`, na samej górze, w sekcji `:root` zmień wartości kolorów.
